# password_generator_Python_version
## Rules
* 密碼規則說明
    1. 密碼不可與前6次重覆
    2. 密碼必須符合複雜性需求
        * 長度至少為 14 個字元
        * 密碼中，不可包含帳號相關字眼
        * 包含下列四種字元中的四種
        * 英文大寫字元 (A 到 Z)
        * 英文小寫字元 (a 到 z)
        * 10 進位數字 (0 到 9)
        * 非英文字母字元 (例如: !、$、#、%)
